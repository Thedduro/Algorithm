# ⚙️ 작은 구슬의 이동 (Medium)

벽으로 둘러싸인 `N × N` 크기의 격자 안에 **하나의 구슬**이 놓여 있습니다.  
이 구슬은 상하좌우 중 특정 방향으로 1초에 한 칸씩 움직입니다.  

가장 왼쪽 위의 칸을 `(1행 1열)`, 가장 오른쪽 아래 칸을 `(N행 N열)`이라 할 때,  
구슬이 벽에 부딪히면 **즉시 방향이 반대로 바뀌며**,  
**방향을 바꾸는 데는 1만큼의 시간**이 소요된다고 합니다.

---

## 📥 입력

- 첫 번째 줄: 두 정수 `N`, `T`  
  - `N`: 격자의 크기 (2 ≤ N ≤ 50)  
  - `T`: 시뮬레이션할 시간 (1 ≤ T ≤ 100)

- 두 번째 줄: 구슬의 초기 상태  
  - `R C D`  
  - `R`: 구슬의 초기 행 번호 (1 ≤ R ≤ N)  
  - `C`: 구슬의 초기 열 번호 (1 ≤ C ≤ N)  
  - `D`: 초기 이동 방향 (`U`, `D`, `L`, `R` 중 하나)

---

## 📤 출력

`T`초 후 구슬의 위치를  
`행 번호 열 번호` 순서로 공백으로 구분해 출력합니다.

---

## 💡 예시

### ✅ 예시 1

**입력**
5 4
1 2 L

**출력**

1 3
