# 📘 문자에 따른 명령 2

좌표평면 위의 로봇이 **북쪽(위)**을 바라본 상태로 `(0, 0)`에서 출발합니다.  
명령어는 다음 세 가지 중 하나로 이루어집니다.

| 명령 | 의미 |
|------|------|
| `L` | 왼쪽으로 90도 회전 |
| `R` | 오른쪽으로 90도 회전 |
| `F` | 바라보는 방향으로 한 칸 전진 |

입력으로 명령어들이 주어지면, **모든 명령을 수행한 뒤의 최종 좌표 `(x, y)`**를 출력하세요.  
단, 시작 방향은 항상 **북쪽(`U`)**입니다.

---

## 🧩 입력
- 첫 번째 줄에 문자열 `dirs`가 주어집니다.  
  (`L`, `R`, `F`로만 이루어짐)

---

## ⚙️ 출력
- 최종 위치 `(x, y)`를 **공백으로 구분하여** 출력합니다.

---

## 🔒 제한 조건
- `1 ≤ 명령의 길이 ≤ 100,000`
